## About Website

This project was built using Javascript, [Bootstrap](https://getbootstrap.com) - a free and open-source CSS framework directed at responsive, mobile-first front-end web development.

### Features

- Responsive website.
- Smooth scrolling.
- Reveal content on scroll.
- Lazy loading images
- Login form.
